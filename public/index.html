Why this matters

The same AI model produces wildly different output depending on prompt specificity. A vague prompt gets a generic page. A detailed prompt gets a production site.

What to include

File structure, design system (colors, fonts, spacing), content sections, responsive breakpoints, SEO metadata, and deployment target. Be specific about what you want.
